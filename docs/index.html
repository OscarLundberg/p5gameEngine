ControlScheme

Control scheme for player objects

new ControlScheme()
Static Members
▸ wasd
wasd
Parameters
directionalOnly (boolean) Only directional buttons. Default = false
horizontalOnly (boolean) Only horizontal directional buttons. Default = false
verticalOnly (boolean) Only vertical directional buttons. Default = false
Returns
any: standard WASD control scheme. This is the default value for Player objects.
Example
{
playerFoo.controlScheme = ControlScheme.wasd();
// W = Directional, Up
// A = Directional, Left
// S = Directional, Down
// D = Directional, Right
// LShift = Button, B
// Space = Button, A
}
▸ arrows
arrows
Parameters
directionalOnly (boolean) Only directional buttons. Default = false
horizontalOnly (boolean) Only horizontal directional buttons. Default = false
verticalOnly (boolean) Only vertical directional buttons. Default = false
Returns
any: arrow keys control scheme
Example
{
playerFoo.controlScheme = ControlScheme.arrows();
// Up = Directional, Up
// Left = Directional, Left
// Down = Directional, Down
// Right = Directional, Right
// LShift = Button, B
// Space = Button, A
}
